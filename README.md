# Vision Soul

一个屏幕桌宠应用 — AI 角色可以看到你的屏幕并和你聊天。

## 快速开始（Windows 用户）

1. 安装 [Node.js](https://nodejs.org/)（下载 LTS 版本，安装时全部默认下一步）
2. 解压本文件夹
3. 双击 `start.bat`
4. 首次运行会自动安装依赖（需要等1-2分钟）
5. 应用启动后，点击灵动岛 → 设置 → 填入你的 API Key

## 快速开始（Mac 用户）

1. 安装 [Node.js](https://nodejs.org/)
2. 打开终端，进入本文件夹
3. 运行 `npm install && npm start`
